# 📊 Excel Data Processing with Python

This project demonstrates how to process and analyze data using Python and pandas.

---

## 🚀 Features
- Reads data from CSV file
- Performs basic data processing
- Generates calculated fields
- Exports processed data to new file

---

## 🛠 Technologies
- Python
- pandas

---

## ▶️ How to run

1. Install dependencies:
```bash
pip install pandas
```

2. Run the script:
```bash
python main.py
```

---

## 📂 Project Structure
- main.py → main script
- data.csv → input data
- output.csv → generated output

---

## 🎯 Purpose
This project demonstrates basic data processing and automation using Python.
